# Online Food Ordering System

## Description
Prototype of an Online Food Ordering System demonstrating SDLC using Jira, Confluence, and GitHub.

## Features
- Browse menu
- Add items to cart
- Place order (simulated)
- Simple frontend prototype using HTML, CSS, JS

## Tools Used
- GitHub: Version control
- Jira: Task tracking
- Confluence: Documentation
- HTML, CSS, JavaScript

## How to Run
1. Open `index.html` in your browser
2. Navigate to Menu → Add items → Cart → Place Order
